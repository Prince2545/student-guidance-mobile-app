# Student Guidance Mobile App

## Description
Student Guidance Mobile App is a React Native app that helps students make better career decisions through guided discovery and structured daily growth.

The app is designed to:
- Help students choose suitable career paths
- Provide daily tasks to build consistent progress
- Track personal progress over time
- Offer AI mentor-style guidance through chat

## Features
- Career discovery flow with 3 guided phases
- Daily task system with proof upload support
- Progress tracking with streak, level, and activity insights
- AI Mentor for chat-based student guidance
- Reset and profile system for personalized use

## Tech Stack
- React Native (Expo)
- TypeScript
- AsyncStorage
- Expo libraries

## How to Run
1. Install dependencies:
   - `npm install`
2. Start the Expo development server:
   - `npx expo start`
3. Open Expo Go on your phone and scan the QR code.

## Folder Structure
Main folders used in this project:
- `src/screens/` - app screens and user flows
- `src/utils/` - utility logic and helper functions
- `src/storage/` - local storage, schema, and persistence logic
- `src/components/` - reusable UI components
